This branch documents the **Forge `1.20.1`** Myotus line.

## What this line ships

- a terminal settings tab backed by `assets/ae2/screens/config/myotus.json`
- a comma hotkey that opens terminal settings while an AE2 terminal screen is focused
- a left-toolbar bug-report button that copies the issue URL to the clipboard
- optional integration markers for JEI, EMI, REI, AE2WTLib, AE2FCT, and AE2TB
- annotation-driven Brigadier command discovery
- a client config file named `myotus-client.toml`

## What this line does not ship

- terminal upgrade cards or persistent upgrade storage
- config tab visibility predicates or tab context helpers
- the newer fluent API aliases added in `1.21.1`
- NeoForge publishing scaffolding

## Recommended path

1. Read [Overview](/myotus/1.20.1/getting-started/overview/).
2. Continue with [Installation](/myotus/1.20.1/getting-started/installation/).
3. Check [Configuration](/myotus/1.20.1/player/configuration/) for the hotkey and client options.
4. Jump to [API Overview](/myotus/1.20.1/developers/api-overview/) if you are integrating another mod.

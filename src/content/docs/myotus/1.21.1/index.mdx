This branch documents the **NeoForge `1.21.1`** Myotus line.

## What this version adds

- static convenience methods on `MyotusAPI`
- fluent registrar aliases
- config-tab visibility and host-context helpers
- NeoForge data conditions through `MyoModCondition`
- JEI, EMI, and REI subscriber hooks
- terminal upgrade cards
- player-persistent terminal upgrade storage
- item registration for dev examples
- Maven publishing support

## Recommended path

1. Read [Overview](/myotus/1.21.1/getting-started/overview/).
2. Continue with [Installation](/myotus/1.21.1/getting-started/installation/).
3. Read [Terminal Upgrade Cards](/myotus/1.21.1/developers/terminal-upgrade-cards/) if you need the new menu extension surface.
4. Use [Migration Notes](/myotus/1.21.1/versions/migration-notes/) when targeting both supported lines.
